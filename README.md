# Smash 'Em

Smash 'Em is a 2D arcade action game developed with TypeScript and the Phaser framework. The goal of the game is to fend off hordes of enemy monsters by jumping on their heads, collect experience points, and become progressively stronger through leveling up.

## Features

* **Endless Survival Mode:** A steadily increasing difficulty through different phases with progressively stronger monster types.
* **Upgrade System:** Rogue-lite mechanics where the player can choose from random upgrades per level up (e.g., higher base damage, critical hit chance, frost attacks).
* **Tactical AI:** Monsters do not just act mindlessly, but adapt their movement behavior based on the player's position in the air or on the ground.

## Controls

* **Movement:** Using Arrow keys or A/D keys to move left and right, Arrow Up, W or Space to jump.
* **Menus & Upgrade Selection:** Mouse click or keyboard (Space/Enter to start, number keys 1, 2, 3 for upgrades).

## Local Development

The project uses Vite as a build tool. Requirements are Node.js and npm.

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
